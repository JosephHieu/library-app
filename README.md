# Library App

A full-stack **Library Management System** built with **Spring Boot** and **ReactJS + TypeScript**.

---


## Tech Stack

### Frontend

* ReactJS
* TypeScript
* React Router
* Stripe Integration

### Backend

* Spring Boot
* Spring MVC
* Spring Security
* MySQL

### Other

* Auth0 (Authentication)
* Stripe (Payment)
* RESTful API
* Microservices-ready structure

---

## Features

### User

* Browse books
* Borrow books
* View book details
* Login / Register (Auth0)
* Ask & answer questions

### Admin

* Manage books (CRUD)
* View user requests
* Respond to user questions
* Manage system data

## Installation & Run

### Clone project

```bash
git clone https://github.com/your-username/library-app.git
cd library-app
```

---

### 2️Run Backend (Spring Boot)

```bash
cd 02-backend/spring-boot-library
./mvnw spring-boot:run
```

---

### 3️Run Frontend (ReactJS)

```bash
cd 03-frontend
npm install
npm start
```

Frontend: http://localhost:3000
Backend: http://localhost:8080
